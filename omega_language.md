# Omega Language

## Variable decleration

var:int hello = 5;

Variable types = integers, character, array, strings

## Variable reference
hello --> Variable reference

## Integers
100_000 : Underscore in integer is ignored

## Strings
`This is a string`
`This is also a stringsakfnfkjsdnfk&5656`

## Arrays
var:str[5] [1, 1, 2, 3, 4] --> Array is all off the same type with pre-defined length


## Mathematics
- + operator for 2 strings, 2 integers, 2 arrays
- - operator for 2 integers
- * operator for 2 integers

## Stdout
out "String"

## Example program

```
var:str String1 = "Hello World!";

out String1;

var:int int1 = 2;

var:int int2 = 4;

var:int int3 = int1 + int2;

var:str string3 = int3;

out string3;
```
